这是一份常见心理问题的自检和应对清单。如果你不熟悉埃格尔自愈指南，请参阅[埃格尔地图](/cognition/2023/08/13/self-cure-map.html)这篇文章。

## 使用方法

1. 先用每一个心理问题的诊断问题部分进行排查，明确自己的症状。
	- **你是不是正在这么想？**
2. 试图思考疗愈问题所提示的方向，进行自我辩论和梳理。
	- **你为什么这样想？**
3. 在问题得到解决后，复盘并优化下一次出现问题时的解决方法。
	- **下次应该怎么想？**

有些问题以陈述句的形式给出。对于这些问题，请问自己“我是不是这样想的？”

## 清单

| 牢笼         | 诊断问题                                                                                                                 | 疗愈问题                                                                                                                                                                                                                                                       |
| ------------ | ------------------------------------------------------------------------------------------------------------------------ | -------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------- |
| 受害者心态   | 为什么是我？<br>他为什么伤害我？<br>我怎么会受到伤害呢？                                                                 | 现在怎么样了?<br>执着于伤害对我有什么好处？<br>我现在还在那种情况下吗？<br>**我对自己负责，每一次伤害都是重新化茧成蝶的机会。**                                                                                                                                |
| 逃避         | 我是不是一直在压抑自己的情绪？<br>我不想让自己有负面情绪，可又做不到。<br>我害怕别人知道我怎么想的。<br>我不能一直这样。 | 不要总说不能，你能做的事有很多。<br>感受有好坏吗？你没有权利感受吗？<br>谁会在乎你的感受，难道不是只有自己吗？<br>为什么要练习逃避感受？为什么不练习感受的能力？<br>这是我的感受还是想法？<br>**我拥有感受的自由。**                                           |
| 自我忽视     | 别人会不会看不起我？<br>别人会不会批评我？<br>别人会不会认可我？                                                         | 我会读心术吗？怎么能知道别人怎么想？<br>别人会读心术吗？怎么能知道我怎么想？<br>真的知道我怎么想的「别人」，是我自己的一部分。<br>我为什么要牺牲自己，为想象的别人殉道呢？<br>我只是别人眼中的标签组成的吗？<br>**我必然是完整的，我的完整映射全世界的自由。** |
| 秘密         | 我这个问题一定得自己解决。<br>我害怕别人知道我这段过往。<br>我要掩盖这个事情。                                           | 我为什么一定要把自己定义在真实和幻想之间？<br>我为什么不愿意尊重真实的自己？<br>我为什么要因为「别人」背负一身秘密？<br>真正的别人会一天到晚惦记我的这些事吗？<br>真正的别人真的在乎这些事吗？<br>**诚实，从对自己说真话开始。**                               |
| 内疚和羞耻   | 我怎么这么没用？<br>我不配。<br>无论我做什么，都不够好。<br>我这个样子没救了。<br>我知道这不对，但我无法改变。           | 我凭什么要定义自己一无是处？<br>我为什么一定要是完美的？这可能吗？<br>我凭什么要为每个人都会有的不完美感到羞耻？<br>我允许自己不完美，我允许自己快乐。<br>**除了我自己，没人会拒绝我。**                                                                       |
| 未解决的悲伤 | 我是否对过去的某件事感到极度愤怒？<br>如果当初是那样的，我就绝不会这样。<br>我无法接受这件事的结果。                     | 我为什么妄图控制过去？<br>现在的我为什么要对无法更改的过去负责？<br>我为什么老念叨“如果”？<br>我为什么不能感受悲伤？<br>**时间捆绑万物，唯有我们自由前行。**                                                                                                   |
| 僵化思维     | 我必须证明我是正确的。<br>他做的不对，他应该这样做。<br>他为什么就不明白呢？                                             | 谁可以是绝对正确的？<br>我为什么要把自己的选择强加于人？<br>我的期待无法满足，是他的错吗？<br>我为什么需要向别人证明自己？<br>**除了黑白，我们拥有看到五彩斑斓的自由。**                                                                                       |
| 怨恨         | 他之前不是这样的。<br>我无法容忍他这些毛病。<br>对方总是朝我大喊大叫。                                                   | 这是谁的问题呢？<br>他应该是幻想中的形象吗？<br>你会和自己结婚吗？<br>**她不再是故事中的人了。**                                                                                                                                                               |
| 恐惧         | 我害怕到无法做出行动。<br>我觉得过去的悲剧会重演。<br>我不能、我尝试、我需要。                                           | 现实真会按照想象发生吗？<br>现实和历史一模一样吗？<br>面对恐惧就会消除恐惧。<br>我不是历史的奴隶。<br>我可以、我想要、我愿意、我选择。<br>我没必要携带继承自他人的恐惧。<br>**[克服恐惧的咒语](no-fear.html)**                                                             |
| 评判         | 他肯定是那样的。<br>他就是那种人。<br>我恨那种人。                                                                       | 我为什么要把世界简化成一棵二叉树？<br>**我可以选择纳粹，也可以选择甘地。**                                                                                                                                                                                     |
| 绝望         | 活着有什么意思？<br>干这些事有什么意思？<br>我没有动力活着。                                                             | 但我没有死，我的理由是什么？<br>我真的可以预测未来吗？<br>我的希望是对世界的一种认知。<br>**这是我的手，血液在其中流淌。**                                                                                                                                     |
| 不宽恕       | 我有一种沉默的愤怒。<br>我绝不原谅这件事。                                                                               | 释放愤怒，接近宽恕。<br>我可以怎么合理地发泄愤怒？<br>我为什么要恐惧愤怒？谁不会发怒？<br>我为什么不能原谅自己？<br>**世界是温柔的，她给你宣泄的自由。**                                                                                                       |
